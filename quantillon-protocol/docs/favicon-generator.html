Quantillon Protocol Favicon Generator

This page can be used to generate a PNG favicon from the SVG.

To generate the PNG favicon:

  1. Right-click on the SVG above
  2. Select "Save image as..."
  3. Save as "favicon.png"
  4. Place the file in the docs/ directory

Alternatively, you can use online SVG to PNG converters or browser developer tools to convert this SVG to PNG format.
